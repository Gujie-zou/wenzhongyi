## 工作流程
1. 每周一早上4点自动抓取新文章（定时任务：wenzhongyi_weekly_update）
2. 逐一精读，提取：药名、组成、主治、症状、功效、方解
3. **索引合并**：新文章主治与现有索引核对，相近的合并到原有分类
4. 同一症状下显示多种药可选（如口臭/上火索引下有2-3种药）
5. 更新网站
6. 向用户汇报新文章情况

## 索引归类规则
**核心原则**：症状相近的合并，避免索引爆炸

**归类示例**：
- 口臭、口腔溃疡、胃热 → 合并到「口臭/上火」
- 风寒感冒、怕冷怕风 → 合并到「感冒/风寒」
- 颈椎病、肩周炎、落枕 → 合并到「颈椎/肩周」
- 黄褐斑、雀斑、色斑 → 合并到「皮肤/色斑」

**索引结构**：
```
症状索引 → 多种药方可选
  ├─ 药A（原方/中成药）
  ├─ 药B（原方/中成药）
  └─ 药C（原方/中成药）
```
